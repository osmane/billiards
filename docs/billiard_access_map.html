Vuruş Erişimi (Geometri) + İlk Spin/Hız (Fizik)

Yan görünüm (x–z)

Yüzey erişim haritası (θ–φ)

Masa düzlemi: z=0 (ekvator) Erişilebilir yüzey: yeşil Erişilemez yüzey: kırmızı (ıstaka masaya takılır) Coulomb: |Jt| ≤ μJn
Harita: Yatay eksen φ (−180°→180°), dikey eksen θ (0° tepe, 180° alt kutup). İşaretçi: seçili temas noktası.

Kısa açıklamalar

  • Erişilebilirlik testi: Seçilen θ–φ noktasında, vuruş doğrultusu êcue geriye doğru L_back boyunca izlenir; çizginin z - r_tip ≥ 0 koşulunu sağlaması gerekir (ıstaka masaya çarpmamalı). Ayrıca ê_cue · n < 0 olmalı (yüzeye yaklaşım).
  • Cue-frame: J = Jc·ê_cue, Jn = −J·n (pozitifse topa doğru), Jt = J − Jn n, Coulomb sınırı aktifle ise |Jt| ≤ μJn.
  • Hız/Spin: v⁺ = J/m, ω⁺ = I⁻¹ (r×J), I=2/5 mR².
  • Follow/Draw: Alt temas noktasının hızı: s = v⁺ + ω⁺×(0,0,−R). Yatay cue yönündeki izdüşüm s·ĉ(+/−) → follow/draw eğilimi.
  • Not: Bu araç vuruş anını ve ilk durumu hesaplar; sonraki kayma→yuvarlanma, Magnus ve sıçrama burada yoktur.
